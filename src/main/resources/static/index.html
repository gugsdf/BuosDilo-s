Página estática com documentação rápida dos principais endpoints. Use Postman ou curl para testar. Esta página evita erro de "No static resource ." quando acessar "/".

Base

Base URL (exemplo no Render): https://ecommerce-api-p2jw.onrender.com

Produtos

GET /api/produtos

Lista todos os produtos

curl -s -X GET https://{{BASE}}/api/produtos
POST /api/produtos

Cria um produto (JSON)

curl -s -X POST https://{{BASE}}/api/produtos \
  -H "Content-Type: application/json" \
  -d '{"nome":"Notebook","preco":3500.00,"sku":"NB001"}'
GET /api/produtos/{id}

Busca produto por id

curl -s -X GET https://{{BASE}}/api/produtos/1

Consumidores

POST /api/consumidores

Cria um consumidor

curl -s -X POST https://{{BASE}}/api/consumidores \
  -H "Content-Type: application/json" \
  -d '{"nome":"Fulano","email":"fulano@ex.com","senha":"123456"}'
GET /api/consumidores/{id}

Busca consumidor por id

curl -s -X GET https://{{BASE}}/api/consumidores/1

Pedidos / Pagamentos / Entregas

Endereço dos principais recursos (exemplos):

  • POST /api/pedidos — criar pedido
  • POST /api/pagamentos — registrar pagamento
  • POST /api/entregas — agendar entrega

Observações
